Tools & Scripts

Facial Joint Rigging Tool [Python]

  • Tool uses user-created CV curves as guides for building the rig; this makes the tool very flexible and applicable to faces of any kind
  • Can build eyelid rigs and controls
  • Includes a function to store custom facial poses in the channel editor while allowing for individual controls to still be keyframed

Biped Auto-Rigging and Pipeline Tool [Python]

  • Used to automatically generate a rig based on a template rig created by the tool
  • Can add squash and stretch to the arms, legs and spine during construction
  • Includes a function to prepare the rig created with this tool for export to game engines such as Unity and Unreal Engine. Can also be applied to animation files.
